Crawling data from HCME

Mở trang EXPA, dùng bộ lọc các columns theo thứ tự như sau:

Applicant Name -> EP ID -> Nationality -> Home LC -> Home MC -> Opportunity -> Organization -> Applied At -> Phone Number -> Slot

Ấn vào nút để lấy Script

Quay lại trang EXPA, nhấn nút F12, vào tab console và Ctrl+V (paste) Script vừa thao tác sao chép, nhấn Enter

Trên thanh của EXPA sẽ xuất hiện nút Copy Data, Nhấn vào để lấy data
